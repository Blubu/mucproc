Summary
------------------
The mucproc.cls is a document class to support the formatting guidelines for submissions to the German **Mensch und Computer** conference.

Files
------------------

The Bundle consists of:
mucproc.cls
mucfontsize10pt.clo

and the demonstration file
mucproc_demo.tex
